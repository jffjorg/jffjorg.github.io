 Case Number:  21STCV43025    Hearing Date:   September 28, 2023    Dept:  30 REBECCA CALE PERICO vs BONNIE BRAE INVESTMENT SERVICES, LLC, A CALIFORNIA CORPORATIONMotion to Be Relieved as Counsel for PlaintiffRuling:  Motion to Be Relieved as Counsel for Plaintiff is denied without prejudice. Moving party to give notice.Discussion:  Counsel has failed to file the requisite forms pursuant to CRC Rule 3.1362. Specifically, Counsel must file MC-051 and MC-052. Further, this motion is set one month prior to the trial, thus a continuance of the hearing so that Counsel can file and serve the correct forms will be unduly prejudicial to the Client.
